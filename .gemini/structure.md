# Project Structure

- `src/`: Root of the source code.
  - `components/`: Reusable UI components.
    - `Navbar.jsx`: Site navigation.
    - `FuncionarioModal.jsx`: Modal for employee details and editing.
    - `AIChat.jsx`: AI chat interface.
  - `pages/`: Page-level components corresponding to routes.
    - `Dashboard.jsx`: Overview and charts.
    - `Cadastro.jsx`: Employee creation form.
    - `ListaFuncionarios.jsx`: Employee table and search.
    - `Relatorios.jsx`: Reporting and data export.
    - `Config.jsx`: App settings.
  - `services/`: Data and external service logic.
    - `api.js`: Main data service handling CRUD (Local vs API).
    - `ai.js`: AI integration service.
  - `App.jsx`: Main application component and routing.
  - `main.jsx`: Entry point.
  - `index.css`: Global styles.
- `public/`: Static assets (CSV template, SVG icons).
- `dist/`: Build output (generated by Vite).
- `.gemini/`: Project documentation and agent context.
- `API_CONFIG.md`: Details for backend integration.
- `DOCUMENTACAO_CODIGO.md`: Technical breakdown of the codebase.
