# Technology Stack

## Frontend
- **React 19:** UI Library.
- **Vite:** Build tool and dev server.
- **React Router Dom 7:** Client-side routing.
- **Vanilla CSS:** Styling approach.

## Libraries
- **Recharts:** For data visualization in the dashboard.
- **Lucide React:** For consistent iconography.
- **PapaParse:** For CSV parsing and generation.
- **jsPDF & html2canvas:** For generating PDF reports.
- **React Markdown:** For rendering AI responses.

## Persistence
- **LocalStorage:** For local development and demo purposes.
- **REST API (External):** Spring Boot backend (configurable URL).

## Development
- **ESLint:** For code quality and linting.
