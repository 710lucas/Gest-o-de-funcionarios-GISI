# Project Features

## Implemented
- **Dashboard (`/`):** Summary cards (total employees, departments, payroll, average salary), charts (department distribution, salary bands), and top 10 salary ranking.
- **Employee Registration (`/cadastrar`):** Form to add new employees with validation.
- **Employee Listing (`/funcionarios`):** Searchable table of employees with details, edit, and delete options via a modal.
- **Configuration (`/configuracoes`):** Toggle between LocalStorage and API, configure API URL, and database maintenance (clear/reset).
- **Reports (`/relatorios`):** (Inferred from existence of `Relatorios.jsx`) Likely for generating PDF/CSV summaries.
- **CSV Import/Export:** Import data from CSV to replace current database; export current data to CSV.
- **Dual Mode Persistence:** Seamless switching between local and remote data sources.
- **AI Chat:** Interactive chat component for AI-assisted queries.
